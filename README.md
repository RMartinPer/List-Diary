# List Diary ![API](https://img.shields.io/badge/API-15%2B-brightgreen.svg?style=flat) [![License: GPL v3](https://img.shields.io/badge/License-GPLv3-orange.svg)](https://www.gnu.org/licenses/gpl-3.0) [![Release](https://img.shields.io/github/release/RMartinPer/List-Diary.svg)](https://github.com/RMartinPer/List-Diary/releases) [![Platform](https://img.shields.io/badge/platform-android-lightgrey.svg)](https://img.shields.io/badge/platform-android-lightgrey.svg) [![Code](https://img.shields.io/badge/code-java-yellow.svg)](https://www.java.com/en/)
